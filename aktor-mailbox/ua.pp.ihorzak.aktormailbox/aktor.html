fun <I, O> CoroutineScope.aktor(mailbox: Mailbox<I, O>, context: CoroutineContext = EmptyCoroutineContext, block: suspend (O) -> Unit): SendChannel<I>
Content copied to clipboard

Launches new coroutine that is receiving messages from its mailbox channel. The resulting object can be used to send messages to this coroutine. This coroutine mailbox preprocesses incoming messages using specified mailbox instance. Uncaught exceptions in this coroutine close the channel with this exception as a cause and the resulting channel becomes failed, so that any attempt to send to such a channel throws exception.

Return

SendChannel to send messages to the aktor mailbox.

Parameters

I

The type of messages accepted by the aktor mailbox.

O

The type of messages actually handled by the aktor.

mailbox

Aktor mailbox which encapsulates preprocessing of aktor input messages before processing. See Mailbox.

context

Additional to CoroutineScope.coroutineContext context of the coroutine.

block

The coroutine code to handle preprocessed messages.
